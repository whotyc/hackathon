Правила пользования

1. Использование материалов сайта разрешено только в некоммерческих целях.
2. Запрещается копирование, распространение или модификация контента без разрешения администрации.
3. Все пользователи обязаны соблюдать законодательство РФ.
4. Администрация оставляет за собой право вносить изменения в правила пользования.

Соглашение об обработке персональных данных

1. Пользователь дает согласие на обработку своих персональных данных для целей участия в проекте.
2. Персональные данные обрабатываются в соответствии с Федеральным законом №152-ФЗ "О персональных данных".
3. Пользователь может отозвать свое согласие на обработку данных, направив запрос на электронную почту support@memorybook.ru.
4. Администрация обязуется обеспечивать защиту персональных данных пользователей.
